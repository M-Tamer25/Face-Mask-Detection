ğŸ§  Model Architecture
The system uses a two-stage pipeline:

Face Detector: A ResNet-10 SSD (Single Shot Detector) model to locate faces in the frame.

Mask Classifier: A MobileNetV2 base (frozen) with a custom head for binary classification (Mask / No Mask).

Optimizer: Adam (lr=1e-4)

Loss Function: Binary Crossentropy
